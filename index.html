About Me

I'm Sam. I'm about to graduate CU Boulder with a BS in Computer Science. I'm currently working as an intern for SketchUp and will be staying full time after I graduate. I'm working as a back-end software engineer for the data warehouse that stores all the 3d models. I find it fulfilling when I can make an impact to the team I am on. I like increasing the performance of code, creating new features, and implementing features others have made. I also enjoy seeing the back end feature I made be used in a front-end feature. I like this because then it allows users to use my work. In my career I value fulfillment, compensation and meaningful work. I am looking for all these aspects in my career. I don't have any specific industry or company that I want to target in the future. In my free time I like to do mostly outdoorsy stuff that involves mountains. I like all types of biking, skiing, and mountaineering. I like to do all these activities when I travel to new places like Utah and Wyoming. This spring I want to travel back to Utah for some more skiing and mountaineering adventures. I want to be able to continue these activities while I work.
